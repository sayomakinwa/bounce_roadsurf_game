Bounce Roadsurf Game

Bounce Roadsurf Game is a game akin to Google Chrome's T-Rex game where you race it out for maximum possible points!


Game Rules

Simple! Avoid the trees and the dead tree trunks on the road and stay in the game for as long as you can. Aha!


How to Play

Simple! Swerve left and right with the left and right arrow keys on the keyboard. Jump with the up or the spacebar key on the keyboard


Game Options

Game Theme: Dark Light
Game Speed: Slow Intermediate Fast Turbo
